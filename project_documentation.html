Design Choices...

● We need to plot the john snow cholera map and necessary visualizations which need to visible all at once on screen and should be in interactive way.

● Need to select the color based on the datasets. So, first i tried with the traditional colors and then picked from the color brewer which are corectly visualizing the data.

● While developing, there was a change in design like using line graph instead of bar chart.

● I created a buttons like age and gender i.e, gender button shows the deaths of male and female at that particular time period and age button divides total death people for all the age groups.

● Features like selecting particular day so as to see only deaths on that day in both colera map and line chart.

Observations...

● I have observered that the deaths of male and female are almost equal. So, immunity is not gender biased.

● Most of the deaths are occured for age group 80 years and above. Therefore people of this age is mostly effected with the disease.

● Form the line graph, most of the deaths are occured in the time period between 26 August to 5 september. Among all the days september 2nd has highest number of deaths.
